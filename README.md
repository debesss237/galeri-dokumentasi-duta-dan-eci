# Galeri Dokumentasi Duta dan ECI

## Deskripsi
Proyek ini adalah galeri dokumentasi yang menampilkan berbagai informasi dan gambar terkait Duta dan ECI. Tujuan dari proyek ini adalah untuk memberikan akses mudah kepada pengguna untuk melihat dan memahami lebih lanjut tentang Duta dan ECI.

## Struktur Proyek
- `index.html`: Halaman utama yang menampilkan galeri.

## Cara Menggunakan
1. Buka `index.html` di browser web.
2. Jelajahi galeri untuk melihat dokumentasi.
